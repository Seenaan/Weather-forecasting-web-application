About This Application

This weather app was inspired by my desire to create a practical application that helps users access accurate weather information easily.

This project is part of my Portfolio Project for ALX-Africa.

This is a web-based weather forecasting application that fetches and displays weather data for a specified city using the OpenWeatherMap API. It provides current weather conditions, a 7-day forecast, and updates the background based on the weather condition.

Features:

  • Fetches weather data based on city name input.
  • Displays current weather details including humidity, pressure, wind speed, sunrise, and sunset times.
  • Provides a 7-day weather forecast.
  • Dynamically updates the background based on weather conditions.
  • Allows searching for weather data by pressing the "Enter" key or clicking the search button.
